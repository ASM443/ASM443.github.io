Projects

Projects with visuals :

Maze solving with DFS, BFS, and Dijkstra's Algorithm

First, a 2D grid is created to represent a maze. Based on a random generator a tile is either blank or a wall. Each tile stores the value of the surrounding tiles. Then, the user can select between depth-first, breath-first search, or Dijkstra's, algorithm. There isn't any noticeable difference at high speeds, but the different methods become visible at slow speeds. This project was implemented in Java.

Image Manipulation

First, an image is converted in to a 2D array filled with a custom RGB object. A new image is created by looping over the array, modifying the RGB object and placing those new RGB objects into a new array. To change the color scheme of a picture the corresponding RGB value is changed. To acheive the "censor" effect a pixel is selected then all surrounding RGB values are set to the same value as inital pixel, this process repeats throughout the entire image. Similar methods are used to create the other effects. This project was implemented in Java.

Blackjack

Traditional blackjack where the user plays against a dealer. The player starts with $4,500 and can make a custom bet against the dealer. The dealer is set to only "hit" if they are below 16. This project was implemented in Java.

Clear Cell

A fast pace clicker game. Before the game starts the user can select the game speed and board the size(not shown in the video). When a block is clicked all surrounding blocks within 1 tile of the same color are cleared, while new blocks are continuously generated at the top. This project was implemented in Java.

Grid Click

Another fast pace clicker game. Before the game starts the user can select the game speed and the board size(not shown in the video). The user starts with a gird of black tiles. Then a green tile will randomly appear for a short duration, if the user clicks the tile while it's still green it will stay green. If the user doesn't click the tile in time it will turn white. If the user clicks a tile that is black it will turn red. Only red, black, and white tile can turn green. This project was implemented in Java.
